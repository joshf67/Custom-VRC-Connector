    Class ByteConverter

    This class is a one-way converter to byte arrays, it doesn't not have a way to revert this This class is not optimized with generic objects so use DataTokens due to using typeof search which means we can't use switches because it's not const This is class is a combination of two different methods of converting to make generating server requests easier.
